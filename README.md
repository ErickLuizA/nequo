## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/nequo.git
  
  or 

 $ gh repo clone ErickLuizA/nequo
```

## 2. Installation

```shell
# Go into the repository
$ cd nequo

# Install dependencies
$ flutter pub get 

# or just use your IDE
```

## 3. Execution

```shell
$ flutter run

# or just use your IDE
```
